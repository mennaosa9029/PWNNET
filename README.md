## 🛠️ Frequently asked questions

### Is PWNNET safe to use?
Yes. PWNNET acts as a passive observer. It does not send your personal data to external servers. All processing happens locally on your machine.

### Why does the app ask for admin access?
To scan your network, the software must view your hardware settings. Windows requires permission for any tool to access these low-level network components.

### Can I run this on a Mac?
PWNNET currently supports Windows machines only. 

### How often should I check my network?
You can run a scan once a week. This helps you spot new devices that might have joined your network without your knowledge.

## 📘 Troubleshooting tips

If you encounter issues during your first scan, try these solutions:

- Restart your router: A quick reboot fixes most local network conflicts.
- Check firewall settings: Sometimes your Windows firewall blocks new applications. Ensure you permit PWNNET to access the network during the first run.
- Update drivers: Ensure your network card drivers are current through the Windows Device Manager.
- Reinstall: If the interface does not load, uninstall the app, restart your machine, and run the installer again.

## 📋 Best practices for network safety

- Update your router firmware: Most security threats enter through outdated router software.
- Use strong passwords: Change the default password for your Wi-Fi immediately.
- Disable unused ports: If you do not recognize a service running on your network, block it.
- Stay informed: Use the information provided by PWNNET to understand which devices communicate with each other.
